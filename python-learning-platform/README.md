# Python Master - Платформа для изучения Python

## 🎓 Описание

Полноценная интерактивная платформа для изучения языка программирования Python с нуля до профессионального уровня.

## ✨ Возможности

### Для учеников:
- **Подробные уроки** - пошаговое обучение от основ до продвинутых тем
- **Интерактивные тесты** - проверка знаний после каждого урока
- **Практические задания** - закрепление материала на практике
- **Онлайн редактор кода** - пишите и запускайте код прямо в браузере
- **Система прогресса** - отслеживайте свой прогресс обучения
- **Достижения (ачивки)** - получайте награды за успехи
- **Личный кабинет** - персональная статистика и настройки

### Технические особенности:
- 🌐 **Двуязычный интерфейс** - русский и английский языки
- 🌓 **Тёмная и светлая темы** - переключение в один клик
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- 💾 **Локальное хранение** - прогресс сохраняется в браузере
- ⚡ **Быстрая загрузка** - оптимизированный код

## 📚 Структура курса

### Уровень 1: Новичок
1. **Введение в Python**
   - Что такое Python? Установка и настройка
   - Переменные и типы данных
   - Условия и ветвления

2. **Циклы**
   - Цикл for
   - Цикл while

### Уровень 2: Средний
1. **Функции**
   - Создание функций
   - Параметры и аргументы
   - Lambda функции

### Уровень 3: Продвинутый
- Веб-разработка (Django/Flask)
- Анализ данных
- Машинное обучение
- Автоматизация

## 🚀 Запуск

1. Откройте файл `index.html` в любом современном браузере
2. Или используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve
   ```

3. Перейдите по адресу `http://localhost:8000`

## 🎯 Как пользоваться

### Начало обучения:
1. Зарегистрируйтесь или войдите в систему
2. Выберите уровень сложности
3. Пройдите урок:
   - Изучите теоретический материал
   - Посмотрите примеры кода
   - Пройдите тест
   - Выполните практическое задание
4. Получите XP и достижения!

### Функции редактора:
- Пишите код Python в онлайн-редакторе
- Запускайте код и смотрите результат
- Копируйте примеры из уроков
- Сохраняйте свои решения

## 🏆 Система достижений

- **Первые шаги** - пройдите первый урок
- **Ученик** - пройдите 5 уроков
- **Недельная серия** - заходите 7 дней подряд
- **Первый проект** - завершите первый проект
- **Скоростной** - пройдите урок за 5 минут
- **Перфекционист** - получите 100% за тест
- **Книжный червь** - прочитайте все материалы
