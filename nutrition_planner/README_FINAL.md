# 🍽️ Nutrition Planner - Планировщик Питания

Полнофункциональное веб-приложение для планирования питания с возможностью интеграции в Telegram бот и мобильное приложение.

## ✅ Статус работы

**Все компоненты протестированы и работают без ошибок:**
- ✅ Веб-интерфейс (5 страниц) - 200 OK
- ✅ REST API - работает
- ✅ База данных - инициализирована
- ✅ Демо-данные - созданы

## 🚀 Быстрый старт на Windows 10

### Способ 1: Автоматический запуск (рекомендуется)

1. Откройте папку `nutrition_planner`
2. Дважды кликните на файл **`START_WINDOWS_FIXED.bat`**
3. Дождитесь сообщения "Запуск веб-сервера..."
4. Откройте браузер: **http://localhost:8080**

### Способ 2: Ручной запуск через командную строку

```bash
# Перейдите в папку проекта
cd nutrition_planner

# Установите зависимости (первый запуск)
pip install fastapi uvicorn sqlalchemy jinja2 aiofiles python-multipart

# Инициализируйте базу данных с демо-данными
python demo.py

# Запустите веб-сервер
python web/app.py
```

- Автогенерация из плана на неделю
- Группировка по продуктам
- Расчет общей стоимости
- Отметка о покупке

## 🔌 Интеграции

## 💡 Примеры использования

### 1. Создать новый ингредиент
1. Перейдите на страницу "Ингредиенты"
2. Нажмите "Добавить ингредиент"
3. Заполните название, КБЖУ, цену, единицу измерения
4. Сохраните

### 2. Создать рецепт
1. Перейдите на страницу "Рецепты"
2. Нажмите "Добавить рецепт"
3. Введите название и описание
4. Добавьте ингредиенты из списка
5. Укажите количество каждого ингредиента
6. Сохраните - КБЖУ рассчитается автоматически

### 3. Спланировать питание на день
1. Перейдите на страницу "Планировщик"
2. Выберите дату
3. Для каждого приема пищи нажмите "+"
4. Выберите рецепт из списка
5. Просмотрите общий КБЖУ за день

### 4. Получить список покупок
1. Перейдите на страницу "Покупки"
2. Выберите неделю
3. Система автоматически создаст список
4. Распечатайте или сохраните

## ⚠️ Решение проблем

### Ошибка "Python не найден"
- Скачайте Python 3.8+ с https://python.org
- При установке отметьте галочку "Add Python to PATH"

### Ошибка "Module not found"
```bash
pip install fastapi uvicorn sqlalchemy jinja2 aiofiles python-multipart
```

### Ошибка базы данных
```bash
# Удалите старую базу и создайте новую
del nutrition_planner.db
python demo.py
```

### Порт 8080 занят
- Измените порт в файле `web/app.py` (последняя строка):
  ```python
  uvicorn.run(app, host="0.0.0.0", port=8081)
  ```
