# 🍽️ Планировщик питания - Инструкция для Windows 10

## ✅ Быстрый старт

### Шаг 1: Установка Python
Убедитесь, что у вас установлен Python 3.8 или выше:
- Скачайте с https://www.python.org/downloads/
- При установке отметьте галочку "Add Python to PATH"

### Шаг 2: Установка зависимостей
Откройте Command Prompt (cmd) или PowerShell в папке проекта и выполните:

```bash
pip install fastapi uvicorn sqlalchemy jinja2 aiofiles python-multipart
```

### Шаг 3: Запуск приложения

**Вариант 1: Только веб-интерфейс (рекомендуется)**
```bash
cd nutrition_planner
python web/app.py
```

**Вариант 2: Веб + API сервер вместе**
```bash
cd nutrition_planner
python run_combined.py
```

### Шаг 4: Откройте браузер

- **Веб-интерфейс**: http://localhost:8080
- **API документация**: http://localhost:8000/docs (если запущен API)

---

## 🔧 Решение проблем

### Ошибка "ModuleNotFoundError"
Убедитесь, что все зависимости установлены:
```bash
pip install --upgrade pip
pip install fastapi uvicorn sqlalchemy jinja2 aiofiles python-multipart
```

### Ошибка "Port already in use"
Порт 8080 или 8000 уже занят. Используйте другой порт:
```bash
# Для веб-интерфейса (измените port=8080 на port=8081 в web/app.py)
# Или закройте программу, использующую порт
```

### Ошибка "Internal Server Error"
1. Проверьте, что база данных создалась корректно
2. Удалите файл `nutrition.db` и перезапустите приложение
3. Проверьте права доступа к папке проекта

### Браузер не открывается автоматически
Откройте браузер вручную и перейдите на:
- http://localhost:8080

## 📱 Возможности приложения

### ✨ Веб-интерфейс
- **Панель управления** - статистика и календарь недели
- **Ингредиенты** - добавление, редактирование, КБЖУ, цены
- **Рецепты** - создание рецептов, расчет nutrition
- **Планировщик** - 6 приемов пищи, выбор даты
- **Список покупок** - автогенерация из плана на неделю

### 🔌 REST API
Доступно для интеграции с Telegram ботом и мобильным приложением:
- http://localhost:8000/docs - интерактивная документация

## 🎯 Как пользоваться

### 1. Добавьте ингредиенты
1. Перейдите в раздел "Ингредиенты"
2. Нажмите "Добавить ингредиент"
3. Укажите название, КБЖУ, цену за 100г, единицу измерения

### 2. Создайте рецепты
1. Перейдите в раздел "Рецепты"
2. Нажмите "Добавить рецепт"
3. Добавьте ингредиенты и их количество
4. КБЖУ рассчитается автоматически

### 3. Спланируйте питание
1. Перейдите в "Планировщик питания"
2. Выберите дату
3. Для каждого приема пищи добавьте рецепты
4. Смотрите общий КБЖУ за день

### 4. Список покупок
1. Перейдите в "Список покупок"
2. Выберите неделю
3. Получите список всех необходимых продуктов
