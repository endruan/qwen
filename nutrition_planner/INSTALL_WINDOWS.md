# 🚀 Установка и запуск на Windows 10/11

## Шаг 1: Проверка Python

1. Откройте командную строку (Win + R, введите `cmd`, нажмите Enter)
2. Введите команду:
   ```
   python --version
   ```
3. Если версия Python 3.10 или выше - переходите к шагу 2
4. Если Python не установлен:
   - Скачайте с https://www.python.org/downloads/
   - Установите, отметив галочку "Add Python to PATH"

## Шаг 2: Установка зависимостей

В командной строке выполните:
```bash
pip install fastapi uvicorn sqlalchemy jinja2 aiofiles python-multipart
```

## Шаг 3: Запуск приложения

### Вариант A: Двойной клик (Простой)

1. Откройте папку `nutrition_planner`
2. Дважды кликните на файл **`START.bat`**
3. Дождитесь сообщения "Uvicorn running on http://0.0.0.0:8080"
4. Откройте браузер и перейдите на **http://localhost:8080**

### Вариант B: Через командную строку (Продвинутый)

1. Откройте командную строку
2. Перейдите в папку проекта:
   ```bash
   cd путь\к\папке\nutrition_planner
   ```
3. Создайте базу данных с демо-данными:
   ```bash
   python demo.py
   ```
4. Запустите веб-сервер:
   ```bash
   python web/app.py
   ```
5. Откройте браузер: **http://localhost:8080**

## Шаг 4: Проверка работы

Откройте в браузере:
- Главная панель: http://localhost:8080
- Ингредиенты: http://localhost:8080/ingredients
- Рецепты: http://localhost:8080/recipes
- Планировщик: http://localhost:8080/meal-planner
- Список покупок: http://localhost:8080/shopping-list

## ❓ Частые проблемы

### Проблема: "python is not recognized"
**Решение**: Переустановите Python, отметив галочку "Add Python to PATH"

### Проблема: "ModuleNotFoundError: No module named 'fastapi'"
**Решение**: Выполните команду установки зависимостей из Шага 2

### Проблема: "Address already in use"
**Решение**: 
1. Закройте все окна командной строки
2. Откройте новое и запустите заново

### Проблема: Страница не открывается
**Решение**:
1. Убедитесь, что сервер запущен (видно сообщение "Uvicorn running...")
2. Проверьте адрес: http://localhost:8080 (без https)
3. Попробуйте другой браузер
