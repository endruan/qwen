# Nutrition Planner - Планировщик Питания

Полнофункциональное приложение для планирования питания с веб-интерфейсом, REST API, Telegram ботом и возможностью интеграции с мобильным приложением.

## 🌟 Возможности

### Веб-интерфейс (Личный кабинет)
- **Панель управления** - статистика, календарь на неделю, быстрые действия
- **Ингредиенты** - база ингредиентов с КБЖУ и ценами, поиск, редактирование
- **Рецепты** - управление рецептами, добавление ингредиентов, расчет КБЖУ
- **Планировщик** - планирование питания на день/неделю/месяц по 6 приемам пищи
- **Список покупок** - автоматическая генерация из плана питания

### REST API
- Полная API документация (Swagger/OpenAPI)
- Интеграция с мобильными приложениями
- Поддержка CORS

### Telegram Бот
- Управление через команды
- Просмотр плана питания
- Быстрое добавление рецептов

## 🚀 Запуск

### Вариант 1: Веб-интерфейс + API (рекомендуется)
```bash
cd nutrition_planner
python run_combined.py
```
Откройте в браузере: http://localhost:8080

### Вариант 2: Только API
```bash
python run_server.py
```
API доступно на: http://localhost:8000
Документация API: http://localhost:8000/docs

### Вариант 3: Telegram бот
```bash
# Установите токен в api/telegram_bot.py
python run_bot.py
```

## 🍽️ Приемы пищи

Приложение поддерживает 6 приемов пищи в день:
1. 🍳 Завтрак
2. 🥐 Второй завтрак
3. 🍲 Обед
4. 🍎 Перекус
5. 🍽️ Ужин
6. 🥛 Второй перекус

## 💡 Основные функции

### Управление ингредиентами
- Добавление/редактирование/удаление
- КБЖУ на 100г
- Цена за 100г
- Единицы измерения
- Поиск по названию

### Управление рецептами
- Создание рецептов
- Добавление ингредиентов с количеством
- Автоматический расчет КБЖУ и стоимости
- Время приготовления
- Количество порций

### Планирование питания
- Выбор даты
- Недельный вид
- Добавление рецептов к приемам пищи
- Расчет КБЖУ за прием пищи и за день
- Расчет стоимости дня

### Список покупок
- Генерация из недельного плана
- Группировка по категориям
- Отметка о покупке
- Подсчет общей стоимости
- Печать списка

## 🔧 Технологии

- **Backend**: Python 3.9+
- **Web Framework**: FastAPI
- **Templates**: Jinja2
- **Database**: SQLite (SQLAlchemy ORM)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Telegram Bot**: aiogram
- **API Docs**: Swagger UI / ReDoc

## 📱 Интеграция с мобильным приложением

REST API предоставляет endpoints для:
- Получения списка ингредиентов и рецептов
- Управления планом питания
- Генерации списка покупок
- Расчета КБЖУ

Пример запроса:
```bash
GET http://localhost:8000/meal-plans/2024-01-15/
```
