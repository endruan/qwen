# PyLearn Platform - Интерактивная платформа для изучения Python

Современная образовательная веб-платформа для изучения Python с нуля до продвинутого уровня.

## 🚀 Особенности

- **Интерактивное обучение** - 200+ уроков с теорией и практикой
- **Онлайн-редактор кода** - запуск Python кода прямо в браузере
- **Геймификация** - достижения, серии дней, прогресс
- **Личный кабинет** - отслеживание прогресса и статистики
- **Две языковые версии** - Русский и Английский
- **Тёмная/Светлая тема** - комфортное обучение в любое время

## 📋 Требования

- Docker & Docker Compose
- Node.js 18+ (для локальной разработки frontend)
- Python 3.11+ (для локальной разработки backend)

## 🛠️ Технологический стек

### Backend
- **FastAPI** - современный асинхронный фреймворк
- **PostgreSQL** - основная база данных
- **SQLAlchemy** - ORM
- **Redis** - кэширование и очереди задач
- **Celery** - выполнение кода пользователей
- **bcrypt** - хеширование паролей
- **PyJWT** - JWT токены

### Frontend
- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - типизация
- **TailwindCSS** - стилизация
- **Zustand** - управление состоянием
- **Recharts** - графики и визуализация
- **Monaco Editor** - редактор кода

### Инфраструктура
- **Docker & Docker Compose** - контейнеризация
- **Nginx** - reverse proxy
- **Makefile** - автоматизация

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
cd pylearn-platform
```

### 2. Настройка переменных окружения

Скопируйте файл `.env.example` и заполните значения:

```bash
cp .env.example .env
```

### 3. Запуск через Docker

```bash
# Сборка и запуск всех сервисов
docker-compose up --build

# Или в фоновом режиме
docker-compose up -d --build
```

### 4. Применение миграций

```bash
docker-compose exec backend alembic upgrade head
```

### 5. Доступ к приложению

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs
- **Admin Panel**: http://localhost:3000/admin

## 📚 Структура курса

Курс состоит из 12 модулей и 200+ уроков:

1. **Основы Python** - введение, установка, первый код
2. **Переменные и типы данных** - числа, строки, списки
3. **Условия и циклы** - if/else, for, while
4. **Функции** - определение, параметры, return
5. **Работа с файлами** - чтение, запись, контекстные менеджеры
6. **ООП** - классы, объекты, наследование
7. **Исключения** - обработка ошибок
8. **Модули и пакеты** - импорт, создание модулей
9. **Работа с API** - HTTP запросы, REST
10. **Базы данных** - SQL, SQLite, PostgreSQL
11. **Асинхронность** - asyncio, await/async
12. **Практические проекты** - реальные приложения

## 🎮 Система достижений

- 🏆 **Первый вход** - за первый вход на платформу
- 📚 **Начинающий** - за прохождение 10 уроков
- 📖 **Читатель** - за прохождение 50 уроков
- 🎓 **Эксперт** - за прохождение 100 уроков
- 🔥 **Серия** - за ежедневное обучение (7, 14, 30 дней)
- 🏅 **Выпускник** - за завершение всего курса

## 🔧 Разработка

### Backend разработка

```bash
cd backend
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend разработка

```bash
cd frontend
npm install
npm run dev
```

## 📝 Тестирование

```bash
# Backend тесты
docker-compose exec backend pytest

# Frontend тесты
docker-compose exec frontend npm test
```
