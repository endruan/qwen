# 🍏 Nutrition Planner - Планировщик питания

Полнофункциональное веб-приложение для планирования питания с личным кабинетом, расчетом КБЖУ и списком покупок.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Создание базы данных с демо-данными
```bash
python demo.py
```

### 3. Запуск веб-сервера
```bash
cd web
python app.py
```

### 4. Открыть в браузере
```
http://localhost:8080
```

## ✨ Возможности

### 📊 Панель управления
- Отображение КБЖУ за день с прогресс-барами
- Расчет остатка калорий и макронутриентов
- Приемы пищи с разбивкой по времени
- Добавки (витамины, микроэлементы)
- Графики статистики

### 👤 Личный кабинет
- Вес и замеры тела (талия, грудь, бицепс, бедра)
- Целевые показатели КБЖУ
- Расчет ИМТ и рекомендации
- График изменения веса
- История замеров

### 🥕 Ингредиенты
- База ингредиентов с КБЖУ
- Учет упаковки и цены
- Автоматический расчет стоимости
- Единицы измерения (г, кг, шт, мл, л)

### 🍳 Рецепты
- Создание рецептов с фото
- Добавление нескольких ингредиентов
- Расчет КБЖУ на порцию
- Поддержка HTML описания
- Количество порций

### 📅 Планировщик
- Планирование на любой день
- 6 приемов пищи
- Недельный обзор
- Навигация по датам
- Добавление/удаление блюд

### 🛒 Список покупок
- Автогенерация из плана на неделю
- Расчет общей стоимости
- Экспорт в файл
- Копирование для Telegram
- Советы по покупкам

## 💡 Примеры использования

### Добавление ингредиента
1. Откройте вкладку "Ингредиенты"
2. Нажмите "+ Добавить ингредиент"
3. Укажите название, КБЖУ, цену и упаковку
4. Сохраните

### Создание рецепта
1. Откройте вкладку "Рецепты"
2. Нажмите "+ Добавить рецепт"
3. Введите название и описание
4. Добавьте ингредиенты с весом
5. Загрузите фото (опционально)
6. Укажите количество порций
7. Сохраните

### Планирование питания
1. Откройте "Планировщик"
2. Выберите дату
3. Для каждого приема пищи нажмите "+ Добавить блюдо"
4. Выберите рецепт и укажите порции
5. Просмотрите итоговое КБЖУ дня

### Генерация списка покупок
1. Составьте план на неделю
2. Откройте "Список покупок"
3. Система автоматически рассчитает продукты
4. Скачайте файл или скопируйте для отправки

## ⚙️ Настройка

### Изменение порта
В `web/app.py` измените:
```python
uvicorn.run(app, host="0.0.0.0", port=8080)
```

### Сброс базы данных
Удалите файл `nutrition.db` и запустите `demo.py` заново.
